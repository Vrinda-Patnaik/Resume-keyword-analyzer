# Resume Keyword Analyzer
An NLP-powered web application that analyzes the match between a resume and a job description, helping users optimize resumes for Applicant Tracking Systems (ATS).

## ðŸ›  Tech Stack
- Python
- Streamlit
- Basic NLP (Text preprocessing & keyword extraction)
- Set-based similarity logic

## How It Works
- Accepts resume and job description as input.
- Preprocesses text (lowercasing, tokenization).
- Extracts keywords from both inputs.

Computes:
- Common keywords
- Missing keywords
- Match percentage
Displays results in an interactive UI.

## Features
- Real-time keyword analysis
- Dynamic match score visualization
- Clean interactive UI
- Session state management
- Modular backend logic (utils.py)
